# MeTTa Data Structures

This project implements list and  basic binary tree data structures in MeTTa:

## 📌 Features

### List
- length
- append
- map
- filter
- max-value
-min value
- foldl / foldr
- sort (insertion sort)
- remove duplicates
-remove elemet

### Binary Tree
- get-root
- get-left / get-right
- size
- search (is-member)


##  Run

```bash
cargo run tree.metta
